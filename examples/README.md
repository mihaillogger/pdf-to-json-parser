# Примеры (sample PDF + ожидаемый JSON)

Демонстрационные прогоны парсера на реальных статьях разных издателей.
Для каждой статьи лежит исходный `<имя>.pdf`, результат `<имя>.json` (объект
`Document` по схеме ТЗ) и кропы фигур/таблиц в `images/<имя>/`.

| Статья | Издатель | DOI | Итог |
| --- | --- | --- | --- |
| `Статья 3` | Springer (J. Radioanal. Nucl. Chem.) | 10.1007/s10967-025-10065-7 | ✅ полный: метаданные + abstract + 14 фигур с подписями + уравнения |
| `c2tc00123c` | RSC (J. Mater. Chem. C) | 10.1039/c2tc00123c | метаданные + 18 фигур + таблицы (1 распознана в 2D) |

Сгенерировано в онлайн-режиме (CrossRef + локальная LLM через Ollama):

```bash
uv run python -m parser --input <pdf> --output examples/
```

> Метрики качества и известные ограничения — в корневом
> [`README.md`](../README.md) и [`evaluation/README.md`](../evaluation/README.md).
