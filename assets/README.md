# assets

Статические изображения для README (скриншоты работы и т. п.).

## Скриншот веб-интерфейса

README ссылается на файл **`assets/streamlit_ui.png`**. Чтобы он отобразился:

1. Поднять интерфейс: `docker-compose up --build -d` → открыть http://localhost:8501.
2. Загрузить любой PDF, дождаться результата.
3. Сделать скриншот и сохранить **строго под именем** `assets/streamlit_ui.png`.
4. Закоммитить картинку (PNG в `assets/` не игнорируется git'ом).
